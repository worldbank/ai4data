# @ai4data/core

Core JavaScript/TypeScript utilities for the **AI for Data – Data for AI** program. Published under the [@ai4data](https://www.npmjs.com/org/ai4data) npm organization.

## Installation

```bash
npm install @ai4data/core
# or
yarn add @ai4data/core
# or
pnpm add @ai4data/core
```

## Development

From the repo root (with workspaces):

```bash
npm install
npm run build --workspace=@ai4data/core
```

Or from this directory:

```bash
cd packages/ai4data/core
npm install
npm run build
npm test
```
